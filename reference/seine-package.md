# seine: Semiparametric Ecological Inference

Efficient and user-friendly routines for modern ecological inference.
Implements the methods described in McCartan & Kuriwaki (2025)
[arXiv:2507.XXXXX](https://arxiv.org/abs/2507.XXXXX), which generalize
ecological regression as introduced by Goodman (1953)
[doi:10.2307/2088121](https://doi.org/10.2307/2088121) . Includes
routines for preprocessing, synthetic data generation, double/debiased
machine learning (DML) estimation, and sensitivity analysis.
